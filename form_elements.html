INPUT

The <input> tag specifies an input field where the user can enter data.

EXAMPLE

            <input type="checkbox" id="checkbox" name="checkbox"/>

SELECT

The <select> element is used to create a drop-down list.

EXAMPLE

            <select name="colors" id="colors">
                <option value="azul">Azul<option>
                <option value="amarillo">Amarillo<option>
                <option value="rojo">Rojo<option>
                <option value="verde">Verde<option>
                <option value="negro">Negro<option>
            </select>



TEXTAREA

The <textarea> tag defines a multi-line text input control.

EXAMPLE

            <textarea id="textarea" name="textarea" rows="4" cols="50">A new revolutionary model 
            to create school of the future and train top software engineers.</textarea>



BUTTON

The <button> tag defines a clickable button.

            <button id="button" type="button">Click to win the lottery!!<button>

FIELDSET

The <fieldset> tag is used to group related elements in a form.

EXAMPLE

            <fieldset id="fieldset" name="fieldset" style="width: 15%;">
                <legend><Personal Details><legend>
                <label for="fname">First Name: <label>
                <input type="text" id="fname" name="fname">
                <label for="lname">Last Name: <label>
                <input type="text" id="lname" name="lname">
                <label for="tel">Number Phone: <label>
                <input type="tel" id="tel" name="tel" placeholder="+34 656 768 443" 
                    pattern="+[0-9]{3}-[0-9]{3}-[0-9]{4}">
            </fieldset>

DATALIST

The <datalist> tag specifies a list of pre-defined options for an <input> element.

EXAMPLE

            <input list="carbrands" name="carbrands" id="carsbrands"/>
            <datalist id="carbrands">
               <option value="VW">
               <option value="Audi">
               <option value="Mercedes">
               <option value="Toyota">
               <option value="Renault">
            </datalist>

OUTPUT

The <outputt> tag is used to represent the result of a calculation (like one performed by a script).

EXAMPLE

            <form oninput="x.value=parseInt(a.value)+parseInt(b.value)">
                <input type="range" id="a" value="50">
                +<input type="number" id="b" value="25">
                =<output name="x" for="a b"><output>
            </form>
