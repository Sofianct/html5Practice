Lists element

<ul>: The Unordered List element

The <ul> HTML element represents an unordered list of items, typically rendered as a bulleted list.

Attributes

square

This atribute create dot square in the list

<ul>
<li>Onion</li>
<li>Tomate</li>
<li>Garlic</li>
</ul>
  • Onion
  • Tomate
  • Garlic
circle

This atribute create dot cicle in the list.

<ul>
<li>Bier</li>
<li>Wine</li>
<li>Water</li>
</ul>
  • Bier
  • Wine
  • Water
disc

This atribute create dot disc in the list.

<ul>
<li>Orange</li>
<li>Limon</li>
<li>Watermelon</li>
</ul>
  • Orange

  • Watermelon

<ol>: The ordered List element.

The <ol> HTML element represents an ordered list of items, typically rendered as a numbered list.

Attributes

reversed

This Boolean attribute specifies that the list's items are in reverse order. Items will be numbered from high to low.

<ol reversed >
<li>Ricardo</li>
<li>Pablo</li>
<li>Soraia</li>
</ol>
  1. Ricardo
  2. Pablo
  3. Soraia
start

An integer to start counting from for the list items. Always an Arabic numeral (1, 2, 3, etc.), even when the numbering type is letters or Roman numerals. For example, to start numbering elements from the letter "d" or the Roman numeral "iv," use start="6".

<ol>
<li>Monica</li>
<li>Peter</li>
<li>Layla</li>
</ol>
  1. Monica
  2. Peter
  3. Layla
type

Sets the numbering type:

  • a for lowercase letters
  • A for uppercase letters
  • i for lowercase Roman numerals
  • I for uppercase Roman numerals
  • 1 for numbers (default)
<ol type="a" >
<li>HTML</li>
<li>CSS</li>
<li>JavaScript</li>
</ol>
  1. HTML
  2. CSS
  3. JavaScript
