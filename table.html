TABLE

The <table> tag defines an HTML table. An HTML table consists of one <table> element and one or more <tr>, <th>, and <td> elements.



TH

The <th> element defines a table header.


TR

The <tr> element defines a table row.


TD

The <td> element defines a table cell.

EXAMPLE

            <table>
            <tr>
                <th>Name</th>
                <th>Age</th>
                <th>City</th>
            </tr>
            <tr>
                <td>Susan</td>
                <td>29</td>
                <td>New York</td>
            </tr>
            <tr>
                <td>Mark</td>
                <td>35</td>
                <td>London</td>
            </tr>
            <tr>
                <td>Arnaud</td>
                <td>19</td>
                <td>Paris</td>
            </tr>
        </table>
Name Age City
